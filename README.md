# Tip Calculator

A basic Tip Calculator project built with HTML, CSS, and JavaScript. This project is ideal for beginners looking to practice front-end web development skills.

## Features

- Clean and intuitive user interface.
- Real-time calculation of tip based on user input.
- Responsive design for a seamless experience on various devices.

## How to Use

1. Enter the total bill amount.
2. Input the desired tip percentage.
3. Click "Calculate Tip" to see the tip amount.

## Tech Stack

- HTML
- CSS
- JavaScript
